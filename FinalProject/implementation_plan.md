# 채팅 기능 개선 계획서

## 미구현 기능 목록

### 1️⃣ 즐겨찾기 채팅방 (상단 고정)
- 채팅방 목록에서 특정 방을 즐겨찾기 설정
- 즐겨찾기한 방은 목록 최상단에 고정 표시

### 2️⃣ 초대 수락/거절 프로세스
- 초대받은 사용자가 수락/거절 선택 가능
- 수락 시 "OO님이 들어왔습니다" 시스템 메시지 전송

### 3️⃣ 메시지 검색 시 해당 위치로 이동
- 검색 결과 클릭 시 해당 메시지로 스크롤 이동
- 해당 메시지 하이라이트 효과

### 4️⃣ 채팅방 생성 시 타입 선택 모달
- "채팅방 생성" 버튼 클릭 시 모달 표시
- 그룹 채팅 / 개인 채팅 선택 후 다음 단계 진행

---

## 구현 완료된 기능

### 실시간 기능
- ✅ **안 읽은 사람 표시** - 모든 메시지에 unreadCount 표시, 실시간 갱신
- ✅ **공감 실시간 갱신** - 다른 사람이 공감을 추가/제거하면 즉시 반영
- ✅ **타이핑 인디케이터** - 입력 중인 사용자 이름 표시, 여러 사용자 동시 표시 (예: "홍길동, 김철수님이 입력 중...")

### 백엔드
- ✅ WebSocket 실시간 이벤트 전송 (`/topic/chat/room/{roomId}/read`, `/topic/chat/room/{roomId}/reaction`)
- ✅ `ChatTypingDto`에 `senderName` 필드 추가
- ✅ 읽음 상태 업데이트 시 실시간 이벤트 전송

### 프론트엔드
- ✅ WebSocket 구독 및 실시간 UI 업데이트
- ✅ 자신이 보낸 메시지의 unreadCount 유지 로직
- ✅ 타이핑 이벤트 전송 시 사용자 이름 포함
