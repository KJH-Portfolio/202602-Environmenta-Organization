## 1. 초기 데이터 삽입 현황

현재 시스템은 시연을 위해 다음과 같은 기초 데이터를 보유하고 있습니다.

| 테이블 그룹 | 데이터 건수 | 주요 내용 |
|---|---|---|
| **MEMBERS** | 3명 | 관리자 1, 일반 유저 2 |
| **ECO_SHOPS** | 15개 | 서울 지역 제로웨이스트 샵 및 수거함 데이터 |
| **CHAT_ROOMS** | 5개 | 관심사별(환경, 비건, 제로웨이스트) 채팅방 |
| **BOARDS** | 20건 | 환경 정보 공유 및 자유 게시판 데이터 |
| **QUESTS** | 10개 | 데일리 친환경 퀘스트 목록 |

## 🔑 2. 테스트 계정 정보 (Test Accounts)

시연 및 기능 테스트를 위해 준비된 계정입니다.

| 분류 | 아이디 (ID) | 비밀번호 (PWD) | 역할 (Role) | 비고 |
|---|---|---|---|---|
| **관리자** | `admin` | `admin123` | ADMIN | 모든 게시글/채팅방 관리 및 신고 처리 가능 |
| **일반유저** | `user01` | `pass01` | MEMBER | 탄소 포인트 및 에코트리 성장 데이터 보유 |
| **테스트유저**| `tester` | `test1234` | MEMBER | 신규 가입 및 초기 퀘스트 수행 가능 |

> **Security Note**  
> 모든 비밀번호는 `BCrypt` 암호화되어 DB에 저장되어 있으며, 위 정보는 개발/테스트 환경의 원문 정보입니다.

## 🛠️ 3. 데이터 초기화 가이드
- 프로젝트 실행 시 `src/main/resources/data.sql` 스크립트를 통해 위 데이터가 자동으로 주입됩니다.
- DB 재구축이 필요한 경우 Docker 컨테이너 재실행 시 초기화됩니다.
